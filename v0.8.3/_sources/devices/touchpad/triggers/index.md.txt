# Triggers
### Properties
All touchpad triggers have the following additional properties:

:::{list-table}
:header-rows: 1

* - Property
  - Type
  - Description

* - fingers
  - *uint*
  - How many fingers must be present on the input device.

    Equivalent to a ``$fingers == [value]`` condition.
:::
